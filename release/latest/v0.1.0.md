# CrowBar v0.1.0

Initial CrowBar release for Minecraft 1.21.11.

## Added

- Locator bar player name tags.
- Locator bar player skin markers.
- Separate keybinds for toggling name tags, skins, and self-view mode.
- Self-view mode for showing your own locator marker on a filled XP bar.
- HUD cleanup while self-view mode is active.
- Mod Menu config screen.
- CrowBar metadata, icon, homepage, sources, and issue tracker links.

## Requirements

- Minecraft 1.21.11
- Fabric Loader 0.18.4 or newer
- Fabric API 0.141.4 or newer
- Java 21
