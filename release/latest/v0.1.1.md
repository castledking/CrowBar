# CrowBar v0.1.1

Added distance display, skin scaling, team colors, and improved Allium integration with independent HUD rendering.

**Download:** [here](https://modrinth.com/plugin/allium-essentials#download)

## Added

- **Show distance**: Toggle to display the distance (in blocks) to each player on the locator bar. When name tags are enabled, distance appears next to the player name (e.g., "PlayerName 50m"). When name tags are disabled, distance appears centered above the locator dot.
- **Allium Plugin Companion**: When connected to a server running the Allium plugin, CrowBar receives player position data directly from the server, enabling accurate distance tracking even for players beyond entity tracking range. Requires the `allium.crowbar` permission on the server.
- **LAN/Singleplayer Support**: On singleplayer or LAN worlds, CrowBar directly accesses the integrated server to get player positions, bypassing the 256-block entity tracking limit.
- **NPC Filtering**: Automatically filters Citizens2 and ZNPCsPlus NPCs from the locator bar when using the Allium plugin companion.
- **Distance-based skin scaling**: Player skin icons now scale based on distance (9px at close range, 5px at far range), matching vanilla locator bar behavior.
- **Team color support**: Player dots now display team colors from scoreboard teams when using the Allium plugin companion.
- **Independent HUD rendering**: Hidden players (sneaking, invisible, wearing carved pumpkin) are now rendered via an independent HUD callback, ensuring they remain visible even when vanilla's locator bar would be empty.
- **Locator bar background**: Custom locator bar background appears when all players are hidden, providing visual context for the independent rendering layer.

## Changed

- **Force show all**: Now ensures hidden players are always visible via independent HUD rendering. Visible players continue to render via vanilla, while hidden players render via Allium data to prevent duplicates.
- **Skin rendering**: Updated to match upstream headlocatorbar scaling behavior for more consistent visuals. Skins render for visible players via vanilla, and for hidden players via Allium when enabled.
- **Distance text anchoring**: Distance text now anchors to the dot/skin position rather than nametags, ensuring it remains visible even when vanilla rendering is absent.

## Requirements

- Minecraft 1.21.11
- Fabric Loader 0.18.4 or newer
- Fabric API 0.141.4 or newer
- Java 21
