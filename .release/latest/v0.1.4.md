# CrowBar v0.1.4

Hotfix for Allium locator bar disappearing when no players are in view

**Download:** [here](https://modrinth.com/mod/crowbar#download)

## Changed

- **Bar background drawn before entry filter**: The locator bar background is now drawn before the per-entry yaw filter, so the bar remains visible even when no Allium-tracked players are in the current viewing angle. Previously the bar would disappear entirely whenever all players were behind the camera.

## Requirements

- Minecraft 1.21.11
- Fabric Loader 0.18.4 or newer
- Fabric API 0.141.4 or newer
- Java 21
