# CrowBar v0.1.3

Improved text readability for off-center locator bar entries

**Download:** [here](https://modrinth.com/mod/crowbar#download)

## Changed

- **Text fade for off-center entries**: The Allium-restored player renderer now uses two-pass rendering. Non-closest entries are drawn first with `0x70FFFFFF` (44% opacity) text, and the entry closest to screen center is drawn last with `0xFFFFFFFF` (full opacity) text. This mirrors vanilla locator bar behavior where peripheral waypoints are visually de-emphasized.

## What's Fixed

- **Text overlap clarity**: Previously all entries rendered text at full opacity, making off-center labels visually compete with the center entry. Faded peripheral text creates a cleaner visual hierarchy.

## Requirements

- Minecraft 1.21.11
- Fabric Loader 0.18.4 or newer
- Fabric API 0.141.4 or newer
- Java 21
- Allium 0.1.8a or newer (recommended)
