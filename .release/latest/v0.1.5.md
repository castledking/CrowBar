# CrowBar v0.1.5

Golden-angle HSV fallback colors matching the Allium server-side generator

**Download:** [here](https://modrinth.com/mod/crowbar#download)

## Changed

- **Fallback dot color switched to golden-angle HSV**: `getMojangishGeneratedColor()` now generates a hue via `(uuidHash * 137.508°) % 360°` with per-hash saturation and brightness variation, replacing the old fixed 16-color Mojang palette. This matches the exact algorithm Allium's server uses for its fallback color, so a player gets the same dot color whether the server sends a team color or both sides fall back independently.

## Removed

- **`MOJANGISH_DOT_COLORS` palette array** — The static 16-entry color table is gone. The golden-angle formula is self-contained and produces the full hue wheel.

## Requirements

- Minecraft 1.21.11
- Fabric Loader 0.18.4 or newer
- Fabric API 0.141.4 or newer
- Java 21
- Allium 0.1.9a or newer (recommended)
