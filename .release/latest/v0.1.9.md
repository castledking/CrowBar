# CrowBar v0.1.9

Allium/CrowBar locator bar now always shows vertical arrows regardless of distance.

## Fixed
- **Locator vertical arrow range**: removed the arbitrary 48-block distance limit on up/down arrows in the locator bar. Vanilla shows these arrows based on camera pitch angle at any distance, and CrowBar now matches that behavior. Affects both the 1.21.x and 26.1.x renderer paths.
