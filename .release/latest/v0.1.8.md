# CrowBar v0.1.8

External HUD suppression support for Axion and other client overlays.

## Added
- **Render suppression API**: added `codes.castled.crowbar.api.CrowBarRenderEvents` for external mods to hide CrowBar/Allium locator rendering while their own HUD overlays are active.
- **Vanilla locator control**: external callers can choose whether suppression keeps the vanilla locator bar visible or hides it too.
- **Owner-scoped suppression**: multiple mods can independently suppress CrowBar rendering without clearing each other's state.

## Changed
- **Axion overlay compatibility**: CrowBar and Allium locator overlays now disappear when Axion requests suppression, preventing them from drawing over Axion's Alt hotbar and tool menus.
- **Cross-version behavior**: suppression is implemented for both the 1.21.x locator bar path and the 26.1.x locator renderer path.

## Fixed
- Fixed CrowBar/Allium locator entries continuing to render during external HUD overlays.
- Fixed the vanilla locator bar staying visible when an external overlay explicitly asks CrowBar to hide both custom and vanilla locator HUDs.
